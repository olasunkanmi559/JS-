METHODS AND PROPERTIES IN JAVASCRIPT



STRING METHOD:

1. Basic String Manipulation

  • length: Returns the length of the string.
    • let str = "Hello";
    console.log(str.length); // Output: 5
  • toUpperCase() / toLowerCase(): Converts the string to upper or lower case.
    • let str = "Hello";
    console.log(str.length); // Output: 5
  • trim(): Removes whitespace from both ends of the string.
    • let str = " Hello ";
    console.log(str.trim()); // Output: "Hello"

2. Searching and Substrings

  • indexOf(): Finds the first occurrence of a substring.
    • let str = "Hello World";
    console.log(str.indexOf("World")); // Output: 6
  • lastIndexOf(): Finds the last occurrence of a substring.
    • let str = "Hello World World";
    console.log(str.lastIndexOf("World")); // Output: 12
  • includes(): Checks if a string contains a substring (returns true or false).
    • let str = "Hello World";
    console.log(str.includes("World")); // Output: true

ARRAY METHOD:

1. Adding and Removing Elements

  • push(element): Adds one or more elements to the end of an array.
    • let arr = [1, 2];
    arr.push(3);
    console.log(arr); // Output: [1, 2, 3]
  • pop(): Removes the last element and returns it.
    • let arr = [1, 2, 3];
    arr.pop();
    console.log(arr); // Output: [1, 2]
  • unshift(element): Adds one or more elements to the beginning of an array.
    • let arr = [2, 3];
    arr.unshift(1);
    console.log(arr); // Output: [1, 2, 3]

2. Iterating Over Arrays

  • forEach(callback): Executes a function for each element.
let arr = [1, 2, 3];
arr.forEach((num) => console.log(num * 2)); // Output: 2, 4, 6

DATE METHOD

1. Creating a Date Object

  • new Date(): Creates a new Date object with the current date and time.
    • let now = new Date();
    console.log(now); // Current date and time
  • new Date(timestamp): Creates a date from a UNIX timestamp (milliseconds since 1 Jan 1970)
    • let date = new Date(1672531200000); console.log(date); // Output: Date based on timestamp
  • new Date(dateString): Parses a date string.
    • let date = new Date("2024-11-25T10:30:00"); console.log(date);

2. Getting Date and Time

  • getFullYear(): Returns the 4-digit year.
    • let date = new Date(); console.log(date.getFullYear()); // Output: 2024
