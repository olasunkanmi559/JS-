METHODS AND PROPERTIES IN JAVASCRIPT



STRING METHOD:

1. Basic String Manipulation

  • length: Returns the length of the string.
    • let str = "Hello";
    console.log(str.length); // Output: 5
  • toUpperCase() / toLowerCase(): Converts the string to upper or lower case.
    • let str = "Hello";
    console.log(str.length); // Output: 5
  • trim(): Removes whitespace from both ends of the string.
    • let str = " Hello ";
    console.log(str.trim()); // Output: "Hello"

2. Searching and Substrings

  • indexOf(): Finds the first occurrence of a substring.
    • let str = "Hello World";
    console.log(str.indexOf("World")); // Output: 6
  • lastIndexOf(): Finds the last occurrence of a substring.
    • let str = "Hello World World";
    console.log(str.lastIndexOf("World")); // Output: 12
  • includes(): Checks if a string contains a substring (returns true or false).
    • let str = "Hello World";
    console.log(str.includes("World")); // Output: true

ARRAY METHOD:

1. Adding and Removing Elements

  • push(element): Adds one or more elements to the end of an array.
    • let arr = [1, 2];
    arr.push(3);
    console.log(arr); // Output: [1, 2, 3]
  • pop(): Removes the last element and returns it.
    • let arr = [1, 2, 3];
    arr.pop();
    console.log(arr); // Output: [1, 2]
  • unshift(element): Adds one or more elements to the beginning of an array.
    • let arr = [2, 3];
    arr.unshift(1);
    console.log(arr); // Output: [1, 2, 3]

2. Iterating Over Arrays

  • forEach(callback): Executes a function for each element.
let arr = [1, 2, 3];
arr.forEach((num) => console.log(num * 2)); // Output: 2, 4, 6

MATH METHOD:

1. Random Numbers

  • Generate a random decimal between 0 (inclusive) and 1 (exclusive):
    • let randomNum = Math.random();
    console.log(randomNum); // Example: 0.572345
  • Generate a random number in a range (e.g., 1 to 100):
    • let min = 1;
    let max = 100;
    let randomInRange = Math.floor(Math.random() * (max - min + 1)) + min;
    console.log(randomInRange); // Example: 42

OBJECT METHOD:

1. Adding Methods to an Object

  • You can define a method as a property of an object:
    • let person = {
    firstName: "John",
    lastName: "Doe",
    fullName: function() {
    return this.firstName + " " + this.lastName;
    }
    };

    console.log(person.fullName()); // "John Doe"

2. Shorthand Syntax

  • Since ES6, you can use shorthand syntax to define methods:
    • let person = {
    firstName: "Jane",
    lastName: "Doe",
    fullName() {
    return `${this.firstName} ${this.lastName}`;
    }
    };
    console.log(person.fullName()); // "Jane Doe"

NUMBER METHOD

1. toString()

Converts a number to a string:

let num = 123;
console.log(num.toString()); // "123"
console.log((123).toString()); // "123"

2. toFixed()

Formats a number to a fixed number of decimal places:

let num = 123.45678;
console.log(num.toFixed(2)); // "123.46"
console.log(num.toFixed(0)); // "123"
